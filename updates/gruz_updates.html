Грузчик - бот для автоматической добычи денег на лесопилке сервера Fawemc.me.
Работает стабильно, багов не обнаружено.
Скачать его можно по ссылке - gruz.rar.
При установке запускаем его, заходим на сервер и нажимаем клавишу Delete.
После вы можете написать в чат /gruz для активации фарма.
